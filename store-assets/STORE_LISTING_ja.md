# Chrome Web Store 掲載文案（日本語）

ダッシュボードの文字数制限に合わせて短縮・調整してください。

## 短い説明（概要）

TikTokのWebサイトで、見ている動画を±1秒・±0.5秒単位で前後に移動できます。

## 詳しい説明

TikTok（tiktok.com）のフィードでは、スクロールで次々と動画が切り替わります。本拡張機能は、**いま画面の中心付近で再生されている動画**を自動的に推定し、その動画に対して **±1秒** と **±0.5秒** のシークボタンを提供します（画面上の表記は英語の s 表記です）。

操作は画面右側に表示される2つのボタンから行います。細かい編集位置の調整や、気になる箇所の再視聴に便利です。

**注意**

- 本拡張は TikTok の公式アプリではありません。
- TikTok の仕様変更により、一時的に動作が不安定になる場合があります。

## 単一目的（Single purpose）用の文章案

TikTok Web 上で、現在視聴中の動画の再生位置を 0.5 秒および 1 秒単位で前後に移動するためのボタンを提供する。

## 権限に関するユーザー向け説明（必要な場合）

本拡張は tiktok.com を開いたときだけスクリプトを読み込み、ページ内の動画要素の再生位置を変更します。個人情報を収集したり外部サーバーに送信したりはしません。
